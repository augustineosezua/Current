Current is a budgeting app built under the Chicken Labs ecosystem, targeted at students and young adults. Its core feature is a "Safe-To-Spend" number that recalculates in real time with every transaction, giving users a single, clear figure representing how much they can spend without jeopardizing their savings goals or upcoming bills. The stack consists of a Node.js + Express backend, a Next.js web frontend, and a React Native + Expo mobile app planned for later, with PostgreSQL + Prisma for the database, Better Auth for authentication, and Plaid for bank connections.
